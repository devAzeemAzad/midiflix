# MidiFlix

React + Vite app that browses movies/TV using the TMDB API.

## Environment Variables

This project requires a TMDB API key at build time:

- `VITE_TMDB_API_KEY` — TMDB v3 API key

### Local

- Create a `.env.local` file:
	- `VITE_TMDB_API_KEY=YOUR_KEY_HERE`

### Netlify

- Site settings → Environment variables
- Add `VITE_TMDB_API_KEY`
- Redeploy (Vite injects env vars during build)

## Scripts

- `npm run dev`
- `npm run build`
- `npm run preview`
